# 🏠 House Price Prediction Flask App

A machine learning web application that predicts housing prices using a trained Random Forest model.

## 🚀 Features
- Predict house prices based on input features
- Flask-based web interface
- Real-time prediction
- Clean UI

## 🧠 Tech Stack
- Python
- Flask
- Scikit-learn
- Pandas, NumPy
